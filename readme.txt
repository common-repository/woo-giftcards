Sell giftcards on your WooCommerce website.

== Description ==

= Requirements =
1. The plugin requires **WooCommerce** to be installed and activated.
2. WooCommerce **Coupons** should be enabled from WooCommerce Settings.

= How it Works =
1. Add new product as gift card (See Screenshot Below)
2. Default Add to Cart button will be remove from gift card  product, it will add new button "Buy Gift Card"
3. The plugin will add form on gift card page where your customers can add recipient information.
4. After successful gift card purchase and when order status turns to processing, it will generate new coupon of format xxxx-xxxx-xxxx-xxxx and send to recipient.
5. Once email is sent, recipient can click the button in the email, it will redirect recipient to the site and apply coupon immediately.

= More options in <a href="https://ahmadshyk.com/item/woocommerce-gift-cards-pro/" target="_blank">Pro</a> Version =
1. Add Giftcards as variable products.
2. An option to add coupon prefix.
3. Send Giftcard to multiple recipients.
4. Options to edit frontend labels and messages.
4. Fully Customize the Giftcard email content.
5. Multiple time giftcard usage (If Giftcard worth more than the previous order)
6. Shortcode to check remaining amount left on Giftcard.
